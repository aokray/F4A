Fairness For All (F4A)

Machine learning can be complicated - and when you begin discussing why and how machine learning can be unfair, it becomes even more difficult to understand. This is why we made Fairness For All (F4A). Not only is this website's goal to make the fairness in machine learning problem accessible to all, but also to facilitate research on fair machine learning so we can truly have Fairness For All.
